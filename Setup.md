# Install dependencies

```sh
npm install
```

# Run

### <u>Usage</u>:
```sh
    node index.js -u username@mail.dom -p password -t to@mail.dom
```
### <u>Description</u>:
```sh
    -u --username   # outlook mail account
    -p --password   # outlook password account
    -t --to         # recipient mail
```
